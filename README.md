# Income Data Analysis Tool

## Overview
A **MATLAB** application that compares average UK salary trends for *graduates*, *postgraduates*, and *non-graduates*.
Features a simple user interface for selecting education groups and automatically visualises key differences.

## Features

Compares income trends across three groups:
- Graduates
- Postgraduates
- Non-graduates

- Loads and processes salary data from online public sources
- Simple pop-up UI for selecting analysis options
- Automatically generates summary statistics and plots
- Highlights key disparities between education pathways

## Tech Stack

Language: MATLAB
Core functionality: data processing (Excel), visualisation, UI pop-ups

## How It Works

- User selects group(s) using the UI popup
- Script loads and cleans data
- Trends and comparative plots are generated
- Key findings are displayed
